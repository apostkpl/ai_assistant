## AI ASSISTANT WRAPPER

This will be a general purpose AI Assistant, aimed to wrap around Clause or chatGPT, and provide a personalized experience to users' agents.
Backend will be in Python, utilizing SQLAlchemy, Pydantic and Uvicorn. Deployment will be done via Docker or Podman.
Dockerfiles will be added later.
